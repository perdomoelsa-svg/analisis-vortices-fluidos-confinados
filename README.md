# analisis-vortices-fluidos-confinados
Análisis matemático de Navier-Stokes en un dominio cilíndrico para el vórtice forzado. Demuestra cómo el equilibrio entre la aceleración centrípeta y la gravedad genera un perfil de superficie parabólico cuadrático $h(r)$, modelando el decaimiento temporal viscoso y la cascada de Kolmogorov $k^{-5/3}$ hacia la disipación microtermal.
